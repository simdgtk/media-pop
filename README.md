# MoPop
MoPop, c'est un média éducatif pour les jeunes qui a pour mission de décrypter l'actualité en expliquant l'origine et le sens des mots qui la composent.

C'est une "application" web créée avec Laravel, Vue.js, Sass, et Laravel Breeze.

Le site est accessible à [ce lien](http://82.165.165.241/). L'app a été pensée pour du mobile.
